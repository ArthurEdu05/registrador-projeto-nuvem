# registrador-projeto-nuvem

# Documentação

## Uma breve descrição da aplicação web e back-end desenvolvidas pelo grupo.
Este projeto é uma aplicação desenvolvida para o registro e consulta de camisas de seleções nacionais de futebol. O objetivo principal é demonstrar o processo de deploy em ambiente de nuvem utilizando containers Docker.
